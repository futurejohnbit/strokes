# 🚀 快速部署指南

## 最快 3 步部署到 Vercel

### 步骤 1：准备文件
下载这 4 个文件到一个文件夹：
- ✅ index.html（游戏主文件）
- ✅ vercel.json（配置文件）
- ✅ README.md（说明文档）
- ✅ package.json（项目信息）

### 步骤 2：上传到 Vercel
1. 访问 https://vercel.com
2. 用 GitHub 账号登录（没有的话先注册）
3. 点击 "Add New..." → "Project"
4. 拖拽整个文件夹到上传区域
5. 点击 "Deploy" 按钮

等待 30 秒，完成！🎉

### 步骤 3：获取网址
部署完成后会得到一个网址，例如：
`https://stroke-hero-game-xxx.vercel.app`

用这个网址就能访问游戏了！

## micro:bit 连接步骤

### 1. 修改你的 Create AI 项目

打开你的 micro:bit MakeCode 项目，添加这段代码：

**在最开始添加：**
```
当开机时:
    蓝牙 启动 uart 服务
    显示图标 ❤️
```

**在识别完成后添加：**
```
当 [AI识别完成]:
    如果 识别结果 = "横":
        蓝牙 uart 写入行 "HENG"
        显示字符串 "一"
    
    如果 识别结果 = "竖":
        蓝牙 uart 写入行 "SHU"
        显示字符串 "丨"
    
    如果 识别结果 = "撇":
        蓝牙 uart 写入行 "PIE"
        显示字符串 "丿"
    
    如果 识别结果 = "捺":
        蓝牙 uart 写入行 "NA"
        显示字符串 "丶"
    
    如果 识别结果 = "点":
        蓝牙 uart 写入行 "DIAN"
        显示字符串 "、"
        
    如果 识别结果 = "勾":
        蓝牙 uart 写入行 "GOU"
        显示字符串 "亅"
        
    如果 识别结果 = "横折":
        蓝牙 uart 写入行 "HENGZHE"
        显示字符串 "乛"
        
    如果 识别结果 = "横折勾":
        蓝牙 uart 写入行 "HENGZHEGOU"
        显示字符串 "⺄"
```

### 2. 下载并烧录

1. 点击 MakeCode 的"下载"按钮
2. 把 .hex 文件复制到 MICROBIT 磁盘
3. 等待黄灯停止闪烁

### 3. 配对蓝牙（仅首次）

1. 按住 micro:bit 的 **A+B** 键
2. 同时按下背面的 **RESET** 按钮
3. 看到配对图案后松开所有按钮
4. 在游戏网页点击"连接 micro:bit"

## 常见问题

**Q: 必须用 Chrome 吗？**
A: 是的，Safari 和 Firefox 不支持 Web Bluetooth。手机只能用 Android Chrome。

**Q: 网页能离线用吗？**
A: 不行，需要联网才能加载汉字数据。展示时确保有 Wi-Fi。

**Q: 可以添加更多汉字吗？**
A: 可以！在代码中的 `charactersList` 数组添加即可，例如：
```javascript
const charactersList = ['大', '人', '木', '中', '田', '口', '日', '小', '火', '水'];
```

**Q: 如何更新网站？**
A: 修改文件后，在 Vercel 仪表板点击 "Redeploy" 即可。
