# 🏰 「口」字城堡迷宮 - 卡通設計方案

## 🎭 **主題背景**

### 🌟 項目概念：筆劃俠客 - 魔法漢字冒險
「筆劃俠客」是一款結合傳統中文書法與現代科技的教育遊戲。玩家使用micro:bit作為「魔法毛筆」，通過做出正確的筆劃動作來控制遊戲角色，在由漢字形狀構成的迷宮中冒險。

### 🏰 「口」字城堡故事背景
> 在遙遠的漢字王國中，有一座神秘的「口」字城堡。傳說這座城堡是古代書法大師用魔法筆劃建造的，只有真正掌握「口」字寫法的人才能找到出路。小明在探險時意外闖入了這座城堡，現在他必須按照正確的筆劃順序——「豎、橫、豎、橫」來破解城堡的秘密，才能安全回家！

## 🎮 **遊戲流程設計**

### 🚀 開場動畫
1. **場景介紹**：城堡全景展示 (3秒)
2. **角色登場**：小明從傳送門出現 (2秒)
3. **任務說明**：智慧爺爺解釋規則 (5秒)
4. **準備開始**：micro:bit連接確認 (2秒)

### 🎯 遊戲進行
```
階段1 - 第1筆豎劃：
👦 小明站在起點
💭 提示："向上揮動micro:bit做豎劃！"
✅ 成功：左側城牆發藍光，小明向上移動
❌ 失敗：城牆震動，重新嘗試

階段2 - 第2筆橫劃：
👦 小明到達左上角
💭 提示："向右揮動micro:bit做橫劃！"
✅ 成功：上側城牆發紅光，小明向右移動
❌ 失敗：提示正確動作

階段3 - 第3筆豎劃：
👦 小明到達右上角
💭 提示："向下揮動micro:bit做豎劃！"
✅ 成功：右側城牆發綠光，小明向下移動

階段4 - 第4筆橫劃：
👦 小明到達右下角
💭 提示："向左揮動micro:bit做橫劃！"
✅ 成功：下側城牆發金光，寶箱出現！
```
