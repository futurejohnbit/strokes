# AI 生成迷宮地圖 Prompt 集合 - 筆劃俠客

## 🎯 基礎 Prompt 模板

### 通用迷宮生成 Prompt
```
請為「筆劃俠客」遊戲設計一個基於中文字「[字]」的迷宮地圖。

要求：
- 迷宮大小：10x10 格子
- 迷宮牆壁要形成「[字]」的筆劃形狀
- 玩家從左上角 (1,1) 開始
- 出口在右下角 (10,10)
- 玩家需要按照正確的筆劃順序移動才能到達出口
- 每個筆劃路徑用不同顏色標示：橫(紅)、豎(藍)、撇(綠)、捺(黃)、點(紫)

輸出格式：
1. ASCII 藝術迷宮圖
2. 筆劃順序說明
3. 移動路徑描述

字形：[字]
筆劃順序：[筆劃列表]
```

## 📝 具體字形示範 Prompts

### 1. 「人」字迷宮

請為「行行出狀元」遊戲設計一個基於中文字「人」的迷宮地圖。

遊戲背景：小明被困在「人」字形的古老迷宮中，需要用micro:bit做出正確的筆劃動作來逃脫。

迷宮設計要求：
- 大小：8x8 網格
- 起點：左上角 (1,1) 標記為 🚪
- 終點：右下角 (8,8) 標記為 🏰
- 牆壁形成「人」字的撇捺結構
- 第一條路徑：撇劃路線（從上到左下）用 ╱ 表示
- 第二條路徑：捺劃路線（從中間到右下）用 ╲ 表示
- 玩家必須先走撇劃路徑，再走捺劃路徑

請提供：
1. ASCII迷宮圖（用 █ 表示牆壁，空格表示通道）
2. 筆劃動作提示：「先做撇劃動作 ↙，再做捺劃動作 ↘」
3. 故事文字：「小明需要模仿『人』字的寫法才能逃出迷宮！」
```

### 2. 「大」字迷宮
```
設計「大」字主題的迷宮關卡：

遊戲情境：勇士被困在巨大的「大」字迷宮神殿中。

迷宮規格：
- 尺寸：10x10 格子
- 起點：頂部中央 (5,1) 🚪
- 終點：底部中央 (5,10) 🏰
- 路徑設計：
  * 第1筆：橫劃 - 從 (3,2) 到 (7,2) 水平通道
  * 第2筆：撇劃 - 從 (5,3) 到 (2,8) 斜向左下通道  
  * 第3筆：捺劃 - 從 (5,3) 到 (8,8) 斜向右下通道

特殊機制：
- 玩家必須按筆劃順序：橫→撇→捺
- 走錯順序會觸發陷阱，回到起點
- 每完成一筆劃，該路徑會發光提示

請生成：
1. 詳細迷宮地圖
2. 筆劃動作指令
3. 遊戲提示文字
```

### 3. 「木」字迷宮
```
創建「木」字森林迷宮：

背景故事：小紅帽迷失在魔法森林中，樹木排列成「木」字形狀。

迷宮特色：
- 大小：12x12 網格
- 環境：森林主題，用 🌲 表示障礙
- 起點：(6,1) 森林入口 🚪
- 終點：(6,12) 森林小屋 🏠

「木」字結構：
- 第1筆：橫劃 - (4,4) 到 (8,4) 
- 第2筆：豎劃 - (6,2) 到 (6,8) 主幹道
- 第3筆：撇劃 - (6,6) 到 (3,9) 左樹枝
- 第4筆：捺劃 - (6,6) 到 (9,9) 右樹枝

遊戲機制：
- 必須按正確筆順移動
- 每個筆劃對應不同的micro:bit動作
- 完成後獲得「森林守護者」稱號

輸出內容：
1. 森林迷宮地圖（用樹木和路徑符號）
2. 筆劃動作教學
3. 故事劇情文字
```

### 4. 「火」字迷宮
```
設計「火」字火山迷宮：

冒險設定：探險家需要穿越火山內部的「火」字形熔岩通道。

迷宮設計：
- 規模：9x9 網格
- 主題：火山熔岩，用 🔥 表示危險區域
- 起點：(5,1) 火山口 🌋
- 終點：(5,9) 安全出口 🚪

「火」字路徑：
- 第1筆：點 - (5,2) 小火苗起點
- 第2筆：撇劃 - (5,3) 到 (3,6) 左側火焰
- 第3筆：捺劃 - (5,3) 到 (7,6) 右側火焰  
- 第4筆：豎劃 - (5,4) 到 (5,8) 中央火柱

安全機制：
- 玩家需要快速完成筆劃動作
- 停留太久會被熔岩燙傷
- 正確筆順可以熄滅部分火焰

請提供：
1. 火山迷宮布局圖
2. 緊急逃生筆劃指南
3. 驚險故事情節
```

## 🎨 進階 Prompt 技巧

### 複雜字形 Prompt
```
為多筆劃漢字「[複雜字]」設計高難度迷宮：

挑戰等級：⭐⭐⭐⭐⭐

設計原則：
- 迷宮大小：15x15 超大網格
- 多層結構：地下1樓、地面、空中平台
- 筆劃交叉點設置傳送門
- 時間限制：3分鐘完成
- 隱藏捷徑：發現字形規律可解鎖

特殊元素：
- 移動平台：某些筆劃路徑會移動
- 機關謎題：需要同時做兩個筆劃動作
- 收集道具：筆劃路徑上的能量寶石

請生成完整的關卡設計文檔。
```

```

## 💡 使用建議

1. **選擇合適的字形**：從簡單的「人、大、小」開始
2. **調整難度**：根據玩家年齡修改迷宮大小
3. **測試平衡性**：確保迷宮有解且不會太難
4. **添加故事性**：每個字都有獨特的背景故事
5. **視覺優化**：使用emoji和符號讓迷宮更生動
