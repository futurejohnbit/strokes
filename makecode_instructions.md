# MakeCode 圖形化編程指南

## 🎯 如何使用 MakeCode 製作這個專案

### 步驟 1：開啟 MakeCode
1. 前往 https://makecode.microbit.org
2. 點擊「新專案」
3. 輸入專案名稱：「漢字識別小助手」

### 步驟 2：基本程式結構

#### 🟢 開始時執行（on start）
```
顯示字串 "漢字識別小助手"
顯示字串 "按 A+B 開始"
設定變數 分數 = 0
```

#### 🔵 當按鈕 A+B 同時按下
```
顯示字串 "遊戲開始!"
設定變數 分數 = 0
重複 5 次：
    呼叫函數 "開始新問題"
    等待按鈕輸入
顯示字串 "遊戲結束!"
顯示變數 分數
```

#### 🟡 當按鈕 A 被按下
```
如果 正確答案 = "A"：
    顯示圖示 開心
    播放音調 高音
    分數 = 分數 + 1
否則：
    顯示圖示 難過
    播放音調 低音
```

#### 🟠 當按鈕 B 被按下
```
如果 正確答案 = "B"：
    顯示圖示 開心
    播放音調 高音
    分數 = 分數 + 1
否則：
    顯示圖示 難過
    播放音調 低音
```

### 步驟 3：建立字形圖案

#### 建立函數：顯示「日」字
```
點亮 LED (0,0) 到 (4,0)  // 上橫線
點亮 LED (0,1) 和 (4,1)  // 左右豎線
點亮 LED (0,2) 到 (4,2)  // 中橫線
點亮 LED (0,3) 和 (4,3)  // 左右豎線
點亮 LED (0,4) 到 (4,4)  // 下橫線
```

#### 建立函數：顯示「目」字
```
點亮 LED (0,0) 到 (4,0)  // 上橫線
點亮 LED (0,1) 和 (4,1)  // 左右豎線
點亮 LED (0,2) 到 (4,2)  // 中橫線
點亮 LED (0,3) 和 (4,3)  // 左右豎線
點亮 LED (0,4) 到 (4,4)  // 下橫線
```

### 步驟 4：所需的積木塊

#### 基本積木
- 🟢 **基本** → 顯示字串
- 🟢 **基本** → 顯示圖示
- 🟢 **基本** → 暫停
- 🟢 **基本** → 清除螢幕

#### 輸入積木
- 🔵 **輸入** → 當按鈕 A 被按下
- 🔵 **輸入** → 當按鈕 B 被按下
- 🔵 **輸入** → 當按鈕 A+B 被按下

#### 邏輯積木
- 🟡 **邏輯** → 如果...否則
- 🟡 **邏輯** → 比較 (=)
- 🟡 **邏輯** → 隨機選擇

#### 迴圈積木
- 🟠 **迴圈** → 重複...次
- 🟠 **迴圈** → 當...為真時

#### 變數積木
- 🟣 **變數** → 設定變數
- 🟣 **變數** → 改變變數

#### 音樂積木
- 🎵 **音樂** → 播放音調
- 🎵 **音樂** → 播放旋律

#### LED 積木
- 💡 **LED** → 點亮 LED
- 💡 **LED** → 關閉 LED
- 💡 **LED** → 切換 LED

### 步驟 5：測試和除錯

1. **模擬器測試**：在網頁上的虛擬 micro:bit 測試
2. **下載程式**：點擊「下載」按鈕
3. **上傳到 micro:bit**：將 .hex 檔案拖到 micro:bit 磁碟

### 🎯 小貼士

#### 簡化版本（適合初學者）
- 只使用 2-3 組字對
- 使用內建圖示代替自製字形
- 減少遊戲輪數

#### 進階版本
- 增加更多字對
- 加入計時功能
- 記錄最高分數
- 加入藍牙功能連接手機

### 🔧 常見問題

**Q: LED 圖案不清楚怎麼辦？**
A: 可以使用更簡單的圖案，或用內建圖示代替

**Q: 程式太複雜怎麼辦？**
A: 先做最基本版本，再逐步增加功能

**Q: 如何讓字形更容易辨識？**
A: 使用對比明顯的字對，如「○」vs「□」的概念
