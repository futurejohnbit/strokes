# 行行出狀元 (Stroke Champs) - micro:bit 創新遊戲

## 🎯 遊戲概念
一款結合 micro:bit 動作感測和手機 app 的創新教育遊戲，專門幫助讀寫障礙兒童學習中文字筆劃和筆順。

### 故事背景
小狀元被困在由漢字筆劃組成的迷宮中，邪惡的「錯字怪獸」在後面追趕。玩家必須用 micro:bit 當作「神奇毛筆」，按照正確的筆順揮動，引導狀元沿著筆劃路徑逃脫！

## 🚀 為什麼選擇 micro:bit？

### 獨特優勢
1. **真實書寫體驗**：物理揮動模擬毛筆書寫，強化肌肉記憶
2. **3D 空間感知**：訓練方向感，對筆劃學習至關重要
3. **專注度提升**：雙手協調操作，減少分心
4. **多感官學習**：視覺（手機）+ 觸覺（micro:bit）+ 聽覺（音效）

### vs 純手機 app
- ❌ 手機：只是手指滑動，缺乏書寫感
- ✅ micro:bit：真實手部動作，符合書寫習慣

## 🎮 遊戲機制

### 核心玩法
1. **迷宮探索**：每個漢字的筆劃形成迷宮路徑
2. **動作控制**：用 micro:bit 做出筆劃動作控制角色移動
3. **筆順學習**：必須按正確順序完成筆劃才能逃脫
4. **傳送機制**：每筆終點有傳送門連接下一筆起點

### 動作對應
- **橫劃** (一)：水平向右揮動
- **豎劃** (丨)：垂直向下揮動  
- **撇劃** (丿)：右上向左下斜揮
- **捺劃** (丶)：左上向右下斜揮
- **點劃** (丶)：快速向下點擊

### 關卡設計
- **初級**：人、大、小、口、日
- **中級**：木、本、末、未、目
- **高級**：複雜字如「森」、「品」

## 品牌命名
- 中文名稱：行行出狀元
- 英文名稱：Stroke Champs

命名說明：
- 「行行出狀元」寓意每一筆劃（行）的練習都能成就未來的狀元，結合教育與成就感。
- 「Stroke Champs」直指「筆劃冠軍」，強調通過練習成為筆劃達人，童趣且具激勵性。

備註：名稱仍可微調與優化，英文名為備選；後續可依展會主題與品牌視覺定稿。
